# The Driver API

## Description

The API for a small logistics company to operate all the transportation data including journeys, parcels, and tracking information.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development watch mode 
$ yarn start:dev

# production mode
$ yarn run start:prod
```
